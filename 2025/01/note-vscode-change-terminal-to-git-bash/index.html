　　我目前在 Windows 11 上的 VSCode 開終端機時，預設是 PowerShell，但有些指令語法跟我較熟悉的 Git Bash 不同……；幸好 VSCode 能改這個設定。


修改 VSCode 的預設終端機

  1. 開啟設定
  2. 在搜尋框輸入 terminal windows
  3. 找到「Terminal > Integrated > Profile: Windows」，是下面有下拉選單的那個
  4. 在下拉選單中選擇想要的終端機
    • null：不指定，在 Windows11 上預設是 PowerShell。
    • PowerShell
    • Git Bash
    • Command Prompt：就是 cmd 啦。
