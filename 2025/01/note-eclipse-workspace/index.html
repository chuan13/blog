　　記錄一自己習慣的 Eclipse workspace 設定。


設定 Preference

  • 上方工具列的［Window］->［Preference］

暗色主題

  1. 左方［General］->［Apperance］
  2. 勾選［Enable theming］
  3. 「Theme」下拉選單選擇［Dark］

自動完成（Auto-Complete）

  1. 左方［Java］->［Editor］->［Content Assist］
  2. 勾選「Insertion」中的［Disable insertion trigger except ‘Enter’］
    • 設為只有『Enter』鍵會套用自動完成的內容，不然時常不曉得按了什麼就套用自動完成了。
  3. 「Auto Activation」裡的「Auto activation trigger for Java」，輸入 ._ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz
    • 只要輸入這些字元，都會跳出 Auto-Complete 功能。

JDK

  1. 左方［Java］->［Installed JREs］
  2. 勾選列表中想要的 JRE
    • 若是沒有……
      1. 點擊右方的［Add…］
      2. 在「JRE home」輸入 JDK 的根目錄
        • 不要選擇「bin」資料夾，要選的是根目錄、裡面有 bin、conf、lib 等等資料夾的那一層。
      3. 在「JRE name」輸入能夠辨認這個 JRE 的名稱
      4. 點擊［Finish］
  3. 左方［Java］->［Compiler］
  4. 「JDK Compilance」裡的「Compiler compliance level」，根據剛剛選擇的 JRE 與實際需求選擇對應的 Java 版本
    • 這裡是選擇要以哪個 Java 版本進行 compile。
    • 這裡選擇的版本不能高於選擇的 JRE 的版本。

Maven

Maven Installation

  1. 左方［Maven］->［Installations］
  2. 勾選列表中想要的 Maven
    • 若是沒有……
      1. 點擊右方的［Add…］
      2. 在「Installations home」輸入 Maven 的根目錄
        • 不要選擇「bin」資料夾，要選的是根目錄、裡面有 bin、conf、lib 等等資料夾的那一層。
      3. 在「Installations name」輸入能夠辨認這個 Maven 的名稱
      4. 點擊［Finish］

Maven settings

　　我之所以想設定這個，是因為我有特別設定 Maven Local Repository 的路徑、寫在我使用的 Maven 裡的 settings.xml。

  1. 左方［Maven］->［User Settings］
  2. 在「User Settings」輸入設定檔的路徑
    • 選擇 （根目錄）\conf\settings.xml
  3. 確認「Local Repository」顯示的是自己特別設定的路徑

設定介面

調整視窗尺寸

  • 把左邊的「Project Explorer」寬度調窄、到上方工具列剛好不會換行
  • 把下面的「Console」高度調整到能顯示 4 行

設定「Project Explorer」的 package 顯示設定

  1. 點擊「Project Explorer」字樣右邊工具欄的［View Menu］（垂直的三個點）
  2. ［Package Presentation］選擇［Hierarchical］
    • Hierarchical 會把 package 以階層顯示。
    • 預設的 Flat，所有 package 都會顯示在同一層、並顯示完整的一長串 package 名稱。

儲存 Perspective 設定

  1. 在右上角的目前 Perspective［Java EE］上按『右鍵』
    • 在搜尋放大鏡按鈕右邊
  2. 選擇［Save As…］
  3. 輸入認得的名稱
  4. 點擊［Save］

設定編輯區文字縮放

  1. 點開任一個檔案
  2. 用『Ctrl』+『=』與『Ctrl』+『-』縮放

輸出 Preference 設定

  1. 上方工具列的［File］->［Export］
  2. 選擇［General］->［Preferences］，點擊［Next］
  3. 勾選［Export all］
  4. 在「To preference file」輸入輸出檔案的路徑
  5. 點擊［Finish］

匯入 Preference 設定

  1. 上方工具列的［File］->［Import］
  2. 選擇［General］->［Preferences］，點擊［Next］
  3. 在「From preference file」輸入 epf 檔的路徑
  4. 點擊［Finish］後，重啟 Eclipse
  5. 選擇自己設定好的 Perspective
